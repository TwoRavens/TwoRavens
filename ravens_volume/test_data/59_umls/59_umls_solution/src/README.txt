This solution requires installation of scikit-tensor using the following steps:
1. Download scikit-tensor from https://github.com/mnick/scikit-tensor/archive/master.zip
2. unzip master.zip to extract tmp/scikit-tensor-master
3. pip install -e tmp/scikit-tensor-master

Note: if you do 'pip install scikit-tensor', it installs python2.5 version.
