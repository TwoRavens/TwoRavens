load

called by main Loads all external data in the following order (logic is not included): 1. Retrieve the configuration information 2. Set 'configurations' 3. Read the problem schema and set 'd3mProblemDescription' 4. Read the data schema and set 'dataschema' 5. Read in zelig models (not for d3m) 6. Read in zeligchoice models (not for d3m) 7. Start the user session 8. Call runPreprocess(...) 9. Call readPreprocess(...) 10. Build allNodes[] using preprocessed information 11. Add dataschema information to allNodes (when in IS_D3M_DOMAIN) 12. Call scaffolding() and start up

scaffolding

called after all external data are guaranteed to have been read to completion. this populates the left panel with variable names, the right panel with model names, the transformation tool, and the associated mouseovers. its callback is layout(), which initializes the modeling space
